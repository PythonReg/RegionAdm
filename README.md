# ⚡ Region Helper by Python

Специализированный АХК-скрипт для автоматизации работы администрации на проекте **Region**. Софт разработан для ускорения ответов на репорты, ведения статистики и быстрого взаимодействия с игровыми механиками.

---

## ✨ Основные возможности
*   🚀 **Авто-ответы:** Мгновенные ответы на популярные вопросы (фин-запросы, правила, навигация).
*   📊 **Учет статистики:** Автоматический подсчет ответов за день и неделю.
*   📍 **Быстрые телепорты:** Мгновенное перемещение по ключевым точкам (МВД, ЦГБ, ГИБДД, СМИ).
*   ⚖️ **Выдача наказаний:** Удобный интерфейс для работы с нарушителями через `Punishment.txt`.
*   🔄 **Автообновление:** Скрипт сам проверяет наличие новых версий и обновляется при запуске.

---

## 🛠 Установка и запуск
1. Скачайте последнюю версию скрипта [здесь](ВАША_ССЫЛКА_НА_RAW_AHK).
2. Установите [AutoHotkey v1.1](https://autohotkey.com), если он не установлен.
3. Положите скрипт в отдельную папку и запустите его.
4. В появившемся меню настройте разрешение экрана и ваш Discord.

## ⌨️ Горячие клавиши (по умолчанию)

| Клавиша | Действие |
| :--- | :--- |
| `Alt + Q` | Приветствие + /tp к игроку |
| `Alt + A` | Фраза "Здравствуйте, иду на помощь" |

| `F4` | Меню быстрых телепортов |
| `F5` | Список всех доступных команд |
| `Ctrl + F9` | Быстрая перезагрузка скрипта |
